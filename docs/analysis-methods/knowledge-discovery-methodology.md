## 1. 知識発見（KDD）の方法論

### 1.1 KDDプロセスの5ステップ

Fayyad et al. (1996) が定義したKDD（Knowledge Discovery in Databases）プロセスは、生データから有用な知識を抽出するための構造化された方法論である。

| ステップ | 内容 | 教育データでの具体例 |
|----------|------|---------------------|
| **1. データ選択** | 関連するデータソースの特定と選択 | 入試・テスト・成績・出欠・アンケートから分析対象を選定 |
| **2. 前処理** | クリーニング、欠損値処理、統合 | 年度間の科目名統一、転入生データの扱い、異常値処理 |
| **3. 変換** | 分析に適した形式への変換 | 正規化、離散化（成績をA/B/C/Dに）、集約（学期→年間） |
| **4. データマイニング** | パターン発見アルゴリズムの適用 | クラスタリング、相関分析、決定木、アソシエーションルール |
| **5. 解釈・評価** | 発見されたパターンの意味付けと検証 | 教員との協議、教育的意味の確認、反例の検討 |

**重要**: KDDは反復プロセスである。一度で完結せず、発見→解釈→新たな問い→再探索のサイクルを回す。

### 1.2 CRISP-DMの教育データへの適用

CRISP-DM（Cross-Industry Standard Process for Data Mining）は、KDDをより実務的に体系化した6フェーズモデルである。教育データマイニング研究で最も広く採用されている方法論（KDnuggets調査で約43%のシェア）。

```
┌─────────────────────────────────────────┐
│         1. ビジネス理解                  │
│   「何を知りたいのか？」                 │
│   教育課題の定義 → データマイニング問題へ変換 │
├─────────────────────────────────────────┤
│         2. データ理解                    │
│   初期データ収集、品質確認               │
│   最初の洞察、仮説の形成                 │
├─────────────────────────────────────────┤
│         3. データ準備                    │
│   クリーニング、統合、変換               │
├─────────────────────────────────────────┤
│         4. モデリング                    │
│   分析手法の選択と適用                   │
├─────────────────────────────────────────┤
│         5. 評価                          │
│   結果の教育的意味の検証                 │
├─────────────────────────────────────────┤
│         6. 展開                          │
│   発見の現場への還元                     │
└─────────────────────────────────────────┘
        ↑                    │
        └────────────────────┘  （サイクル）
```

**教育での適用ポイント**: フェーズ1で「生徒の成績を予測する」というデータマイニング問題に変換するのではなく、「**なぜ**特定の生徒がつまずくのか」「何が**介入可能な**要因なのか」という教育課題として定義することが重要。

### 1.3 「仮説駆動型」vs「データ駆動型」の探索

| 側面 | 仮説駆動型（トップダウン） | データ駆動型（ボトムアップ） |
|------|---------------------------|----------------------------|
| **起点** | 理論・教員の経験知から仮説を立てる | データからパターンを自動検出する |
| **方法** | 仮説検定、回帰分析、事前定義した変数間の関係検証 | クラスタリング、アソシエーションルール、異常検知 |
| **強み** | 解釈しやすい、既知の理論体系に位置づけ可能 | バイアスが少ない、意外な発見（セレンディピティ）の可能性 |
| **弱み** | 知っていることしか見つからない | 偽陽性が多い、解釈困難なパターンも出る |
| **使い時** | 教員が「多分こうじゃないか」と思っている仮説がある時 | 「何かありそうだが何かわからない」時 |

**実践的な使い分け戦略**:
1. **第1段階**: データ駆動型EDAで全体像を把握（下記チェックリスト参照）
2. **第2段階**: 発見されたパターンから仮説を生成
3. **第3段階**: 仮説駆動型で厳密に検証
4. **第4段階**: 検証結果から新たな問いを生成 → 第1段階に戻る

この「演繹と帰納の弁証法的プロセス」が、知識発見の本質である。

### 1.4 探索的データ分析（EDA）の実践チェックリスト

```
□ Shape（形状）
  - 行数（生徒数×テスト数）、列数（変数の数）
  - データの期間（何年分？何学期分？）

□ Types（型）
  - カテゴリ変数（性別、コース、クラス）vs 連続変数（点数、出欠日数）
  - 順序変数（成績評定A/B/C/D）の扱い確認

□ Missingness（欠損）
  - 欠損率の全体像と変数別分布
  - 欠損パターンの確認（MCAR/MAR/MNAR）
  - 転入・転出・長期欠席による構造的欠損

□ Distributions（分布）
  - 各変数のヒストグラム（正規？二峰？天井効果？）
  - テスト得点の分布 → 設問の難易度情報として活用

□ Relationships（関係性）
  - 相関行列ヒートマップ → 高相関ペアの発見
  - 教科間相関（数学×理科、国語×英語 etc.）
  - 出欠×成績、アンケート回答×成績の関係

□ Temporal Patterns（時間的パターン）
  - 学期ごとの成績推移
  - 出欠パターンの変化（いつから増えた？）
  - 模試のスコア推移

□ Subgroup Differences（サブグループ差）
  - クラス別、コース別、入試区分別の比較
  - 男女差、入学時成績層別の差

□ Anomalies（異常値・外れ値）
  - 突出した成績変動（急上昇・急降下）
  - 異常な出欠パターン
  - アンケートの矛盾回答
```

### 1.5 セレンディピティ（偶然の発見）を設計する

「意外な発見」は偶然に見えるが、実は**構造的に設計できる**。

**「系統的セレンディピティ」（Systematic Serendipity）の原理**:
天文学の研究（Giles & Walkowicz, 2019）で提案された手法。密度ベースのクラスタリングでデータを分類し、密な領域の外にある4%未満のデータを「外れ値（anomaly）」として検出する。この手法で、従来見逃されていた希少現象が発見された。

**教育データへの応用**:
1. **教師なし学習で外れ値を検出**: 全変数を使ってクラスタリングし、どのクラスタにも属さない生徒を特定 → 「なぜこの生徒は典型的パターンから外れるのか」を調査
2. **異常スコアパターンの検出**: 「数学は上位10%だが国語は下位20%」のような非典型的な能力プロファイルを検出
3. **予測残差の分析**: 予測モデルが大きく外した生徒を調査 → モデルが捉えていない要因の発見
4. **ランダムな変数ペアの可視化**: 意図的に「関係なさそうな」変数同士を散布図で確認

**アソシエーションルールマイニング**: Aprioriアルゴリズムを使い、「IF 出欠率 < 50% AND 宿題提出率低 THEN 期末試験60%未満」のようなルールを自動発見。ある教科の成績が別の教科の成績を予測できるような隠れた関係も検出可能。

## 2. 疫学の視点 -- 教育への転用

### 2.1 なぜ疫学か？

疫学は「集団における健康事象の分布と決定因子を研究し、健康問題の制御に応用する学問」。これを教育に読み替えると：

| 疫学の概念 | 教育への転用 |
|-----------|-------------|
| 疾患 | 学力不振、不登校、中退 |
| リスクファクター | 欠席、低い入試成績、特定のアンケート回答パターン |
| 曝露（Exposure） | 特定の指導法、カリキュラム、環境要因 |
| アウトカム | 成績向上、大学合格、自己効力感の変化 |
| コホート | 入学年度の同期生 |
| インシデンス（発生率） | 学力不振の新規発生率 |
| プレバレンス（有病率） | ある時点で学力不振である割合 |

### 2.2 コホート研究デザインの教育応用

#### 前向きコホート研究（Prospective Cohort）
- **設計**: 入学時点のデータ（入試成績、アンケート）を記録 → 3年間追跡 → 卒業時の成果を測定
- **利点**: 時間的な先行性が確保でき、因果関係の方向が明確
- **教育での実装**: 既にデータが蓄積されているなら、「入学時のデータで3年後を予測するモデル」として分析可能

#### 後向きコホート研究（Retrospective Cohort）
- **設計**: 卒業生のアウトカム（大学合格、成績推移）から遡って入学時の特徴を分析
- **利点**: 既存データで即座に分析開始可能
- **注意**: 「記憶バイアス」はないが、当時測定していない変数は分析不能

**ハザードモデルの活用**: 生存分析（Survival Analysis）の手法を転用し、「学力不振に陥るまでの時間」をモデリング。比例ハザードモデルでリスクファクターの影響度を推定できる。研究例では、1年目の成績（取得単位数）が中退の最も強力な予測因子であることが確認されている。

### 2.3 オッズ比・相対リスク・寄与危険度の算出

#### 2x2表の構成（教育データ版）

```
                    アウトカム
                  不振あり    不振なし
曝露あり            a           b        a+b
（欠席多い）
曝露なし            c           d        c+d
（欠席少ない）
                  a+c         b+d        N
```

#### 指標の計算と解釈

| 指標 | 計算式 | 教育での解釈例 |
|------|--------|---------------|
| **相対リスク（RR）** | [a/(a+b)] / [c/(c+d)] | 欠席が多い生徒は少ない生徒の何倍、学力不振になりやすいか |
| **オッズ比（OR）** | (a*d) / (b*c) | 学力不振群で欠席多の割合と、そうでない群での割合の比 |
| **寄与危険度（AR）** | a/(a+b) - c/(c+d) | 欠席が多い群で、欠席に起因する学力不振の追加割合 |
| **人口寄与危険割合（PAF）** | 全体のリスクのうち、この要因で説明できる割合 | 欠席問題を解消した場合に減少する学力不振の割合 |

**実用上の注意**:
- RRは前向き研究（コホート研究）でのみ算出可能
- 後向き研究（既存データの分析）ではORを使う
- アウトカムが稀（<10%）な場合、ORはRRの良い近似値になる

### 2.4 交絡因子（Confounders）の統制

**交絡とは**: 第三の変数がExposureとOutcomeの両方に影響を与え、見せかけの関連を生み出すこと。

#### 教育データでの典型的な交絡の例

```
  社会経済的背景（SES）
      ↓           ↓
  出欠状況 ──?──→ 成績
```

「欠席が多いと成績が低い」は真の因果関係か、それとも家庭環境（SES）が両方に影響しているだけか？

#### 統制の方法

| 手法 | 説明 | 教育データでの適用 |
|------|------|-------------------|
| **層別分析** | 交絡因子の値ごとに分けて分析 | SES層別（奨学金有無等で代理）に出欠と成績の関係を分析 |
| **マンテル・ヘンツェル法** | 層別ORの加重平均 | 複数の交絡因子を同時に統制 |
| **多変量回帰** | 交絡因子を共変量として投入 | 重回帰分析で出欠の効果をSES統制後に推定 |
| **ANCOVA** | 分散分析に共変量を投入 | クラス間比較でSES・入学時成績を統制 |

#### Simpson's Paradox（シンプソンのパラドックス）に注意

全体では「プログラムAが有効」に見えるが、男女別に分析すると「両方でプログラムBが有効」という逆転が起こりうる。教育データでは、コース別・クラス別に分析結果が逆転していないか必ず確認する。

## 3. 学習科学 -- 何を探すべきか

### 3.1 つまずきの連鎖（Prerequisite Relationships）

学習内容には前提関係がある。「分数ができないと方程式ができない」のような依存関係を明らかにすることで、つまずきの根本原因を特定できる。

**検出方法**:
- 設問単位テストの正答パターンから、「Aが解けない生徒はBも解けない」という含意関係を検出
- 条件付き確率で表現: P(B正答 | A不正答) が極めて低い場合、AはBの前提条件

### 3.2 Knowledge Space Theory（知識空間理論）

Doignon & Falmagne (1985) が創始した理論で、**設問間の前提関係を数学的にモデル化**する。

#### 核心概念

| 概念 | 定義 | 教育データでの意味 |
|------|------|-------------------|
| **知識状態（Knowledge State）** | ある生徒がマスターしている概念の集合 | 正答できる設問の集合 |
| **知識空間（Knowledge Space）** | 取り得るすべての知識状態の集合族 | 生徒が実際に示す正答パターンの全体 |
| **推測関係（Surmise Relation）** | 「aができるならbもできるはず」という半順序 | 前提条件の構造 |
| **学習パス（Learning Path）** | 知識状態を一つずつ拡大する経路 | 効率的な学習順序 |

#### 実装のイメージ

```
知識空間: {}, {q1}, {q2}, {q1,q2}, {q1,q3}, {q1,q2,q3}, {q1,q2,q3,q4}

推測関係:
  q1 は q3 の前提条件（q3 ができるなら q1 もできる）
  q1, q2 は q4 の前提条件
```

**ALEKSシステム**: この理論に基づく実用的な数学評価システム。生徒が「何ができて」「次に何を学ぶ準備ができているか」を特定できる。

**教育データへの適用**: 設問単位テストの正答データから、Item Tree Analysis等の探索的手法で知識空間を構築し、各生徒の知識状態を推定 → 個別に「次に取り組むべき内容」を特定。

### 3.3 認知負荷理論（Cognitive Load Theory）

Sweller (1988) の理論。設問の難しさを3つの負荷に分解する。

| 負荷の種類 | 内容 | データから推定する方法 |
|-----------|------|---------------------|
| **内在的負荷（Intrinsic Load）** | 学習内容自体の難しさ。前提知識に依存 | IRT（項目反応理論）の難易度パラメータ、正答率 |
| **外在的負荷（Extraneous Load）** | 教材の設計による不要な負荷 | 同じ内容で設問形式が異なる場合の正答率差 |
| **本有的負荷（Germane Load）** | スキーマ構築に使われる有効な負荷 | 応用問題への転移率 |

**項目反応理論（IRT）との連携**: IRTの難易度推定値を認知負荷の代理指標として活用できる。研究で、IRTの難易度推定と学習者の主観的認知負荷評定に有意な相関が確認されている。

**実践的示唆**: 設問の難易度が前提知識の有無に依存するなら、「つまずき」は内容が難しいのではなく前提条件の欠如が原因。→ 前提条件の特定が本質的な介入ポイント。

### 3.4 転移（Transfer）

ある教科の学びが別の教科に影響するメカニズム。

| 転移の種類 | 説明 | 教育データでの検出 |
|-----------|------|-------------------|
| **近転移（Near Transfer）** | 類似した文脈間の転移 | 数学Iの成績 → 数学IIの成績への予測力 |
| **遠転移（Far Transfer）** | 異なる文脈間の転移 | 数学の成績 → 物理の成績への予測力 |

**実証研究の知見**:
- 数学の成績は理科系科目の成績の**43～87%**の分散を説明する（極めて強い転移）
- 読解力（国語）は、社会科・理科の文章題の成績に影響する（遠転移）
- 教科横断的な学習を行っている生徒は、全教科で**32%高い**達成度を示す
- メタ認知能力が転移の促進因子として機能する

**データ分析で探るべき問い**:
- どの教科ペアで最も強い相関があるか？（相関行列の分析）
- 相関が時間とともに変化するか？（学年進行に伴う変化）
- 特定のスキル（読解力等）が複数教科に波及するか？（構造方程式モデル）

### 3.5 メタ認知と自己調整学習

アンケートデータから測定可能な認知的・動機的要因。

**主要な知見**:
- メタ認知戦略の使用と読解・数学・理科の成績に正の相関が確認されている（大規模調査）
- 自己調整学習（SRL）のMSLQ（Motivated Strategies for Learning Questionnaire）による測定値と成績の平均相関: r = 0.20
- 非認知要因（学業エンゲージメント、自己効力感、行動調整、誠実性）だけで**GPAの17～25%**の分散を説明
- 過去の成績と合わせると**50%**まで説明可能

**アンケートで測定すべき構成概念**:
| 構成概念 | 内容 | 測定の難しさ |
|---------|------|------------|
| モニタリング | 自分の理解度の把握 | 自己報告バイアスあり |
| 計画 | 学習計画の立案・実行 | 効果は限定的（研究で周辺的効果のみ） |
| 努力調整 | 困難でも粘り強く取り組む | 成績との相関が安定的 |
| 時間管理 | 学習時間の配分 | 成績との相関が安定的 |

### 3.6 非認知能力（Grit, Growth Mindset, Self-efficacy）と学力

**最新のメタ分析知見（2024-2026年の研究）**:

| 構成概念 | 学力との関係 | 効果量 | 特記事項 |
|---------|-------------|--------|---------|
| **Grit（やり抜く力）** | 正の相関 | 中程度 | 「努力の継続」次元が「興味の一貫性」より強い |
| **Growth Mindset（成長マインドセット）** | 正の相関 | 小～中 | Gritと r=0.19（メタ分析, N=42,112） |
| **Self-efficacy（自己効力感）** | 正の相関 | 中～大 | 最も安定した予測因子 |

**重要な発見**: 
- これらの非認知能力は**直接的に**学力を予測するだけでなく、**連鎖的に**影響する（Growth Mindset → Grit → Self-efficacy → 学力）
- 認知能力は10歳頃に安定するが、非認知能力は**青年期以降も可塑性がある** → 高校段階での介入に意味がある
- 介入研究では、Growth Mindset介入が特に学力下位層で効果が大きい

## 4. 統計的検出力と実現可能性

### 4.1 生徒数500-1000人で何が検出可能か

**結論**: 500-1000人は教育研究では**大規模**サンプルであり、多くの分析に十分な検出力がある。ただし、クラスタリング（クラス・コース）の影響を考慮する必要がある。

#### 検出可能な効果量の目安

| 分析の種類 | N=500 | N=1000 | 必要な効果量 |
|-----------|-------|--------|------------|
| 2群間の平均差（t検定） | d=0.18検出可能（80%パワー） | d=0.13検出可能 | 小 |
| 相関係数 | r=0.12検出可能 | r=0.09検出可能 | 小 |
| 回帰分析（10変数） | R²=0.04検出可能 | R²=0.02検出可能 | 小 |
| サブグループ比較（n=100 vs 100） | d=0.40必要 | d=0.28必要 | 小～中 |

### 4.2 効果量（Effect Size）の考え方

#### Cohen's d の基準

| 効果量 | Cohen's d | 教育での意味 | 具体例 |
|--------|----------|------------|--------|
| **小** | 0.2 | 肉眼では見えにくいが存在する差 | アンケート1項目の回答傾向の差 |
| **中** | 0.5 | 注意深く見れば認識できる差 | 出席率上位群と下位群の平均点差 |
| **大** | 0.8 | 明白な差 | 入試上位層と下位層の3年後成績差 |

**重要な注意**: Cohen (1988) の基準は一般的なものであり、教育研究の文脈では分野固有の効果量分布に基づいて解釈すべき（米国教育省IESの勧告）。教育介入の効果量はd=0.1～0.3が現実的であり、d=0.2でも実践的に意味がある場合がある。

#### 統計的有意 ≠ 実践的に重要

- サンプルが大きいと（N=1000）、実質的に無意味な小さな差でもp < 0.05になる
- 逆に、実践的に重要な差がサンプル不足で有意にならないこともある
- **常に効果量を報告し、「教育現場でこの差は意味があるか？」を判断基準にする**

### 4.3 多重比較問題

**問題の本質**: 100回の検定を行えば、真に差がなくても5回は「有意」になる（α=0.05の場合）。教育データで多数の変数をクロス分析すると、偽陽性が大量に発生する。

#### Bonferroni補正

- **方法**: 有意水準を検定数で割る（α' = 0.05 / k）
- **利点**: Family-wise Error Rate（FWER）を厳密に制御
- **欠点**: 非常に保守的。検定数が多いとほとんど何も有意にならない
- **使い時**: 1つでも偽陽性があると深刻な場合（例: 新しい教育政策の根拠にする場合）

#### FDR（False Discovery Rate）制御 -- Benjamini-Hochberg法

- **方法**: p値を小さい順に並べ、「発見」のうち偽陽性の割合をq以下に制御
- **利点**: Bonferroniより検出力が高い。検定数が増えるほど有利
- **欠点**: ある程度の偽陽性を許容する
- **使い時**: 探索的分析で多数のパターンを検討する場合（教育データマイニングの多くの場面）

**Benjamini-Hochberg法の手順**:
1. k個のp値を小さい順に並べる: p(1) <= p(2) <= ... <= p(k)
2. 各p値に対して閾値を計算: threshold(i) = (i/k) * q （q = 許容FDR、通常0.05～0.10）
3. p(i) <= threshold(i) を満たす最大の i を見つける
4. p(1) ～ p(i) に対応する仮説を「発見」とする

**実践的推奨**: 教育データの探索的分析では、FDR制御（q=0.05～0.10）をデフォルトとし、政策判断に関わる重要な結果のみBonferroni補正で確認する。

### 4.4 クラスタリング（階層構造）の影響

教育データには「生徒 → クラス → コース → 学校」の階層構造がある。同じクラスの生徒は、異なるクラスの生徒よりも類似する傾向がある。

**ICC（級内相関係数）**: クラスタ内の類似度を表す指標
- 教育データの学校レベルICCの中央値: 0.031（四分位範囲: 0.011～0.08）
- つまり、成績の分散の約3%は「どの学校（クラス）に属するか」で説明される

**デザイン効果**: 有効サンプルサイズの縮小
- Design Effect = 1 + (m - 1) * ICC
- m = クラス平均人数 = 40, ICC = 0.05 の場合: DE = 1 + 39 * 0.05 = 2.95
- **実効サンプルサイズ = 1000 / 2.95 ≒ 339人分の情報量**

**対策**: マルチレベルモデル（階層線形モデル, HLM）を使用して適切に推定する。

## 5. 因果推論 -- 相関から因果へ

### 5.1 「相関は因果ではない」問題

相関が因果でない理由は3つ:
1. **逆の因果**: AがBを引き起こすのではなく、BがAを引き起こしている
2. **交絡**: 第三の変数CがAとBの両方に影響している
3. **偶然**: 多数の比較で偶然有意になった（多重比較問題）

### 5.2 因果推論のフレームワーク

#### Rubin Causal Model（潜在結果モデル）

| 概念 | 定義 | 教育での例 |
|------|------|-----------|
| **潜在結果（Potential Outcomes）** | 処置を受けた場合/受けなかった場合の両方の結果 | 補習を受けた場合の成績 / 受けなかった場合の成績 |
| **因果効果** | 潜在結果の差 | 補習あり成績 - 補習なし成績 |
| **根本問題** | 一人の生徒で両方は観測不能 | 補習を受けた生徒の「受けなかった場合」は反事実 |
| **解決策** | 集団レベルで平均処置効果（ATE）を推定 | 類似集団を比較して補習の平均的効果を推定 |

#### Judea Pearlの因果推論フレームワーク

**3つの中核要素**:
1. **DAG（有向非巡回グラフ）**: 変数間の因果関係を視覚的に表現
2. **構造因果モデル（SCM）**: 各変数を他の変数の関数として定義
3. **do演算子（do-calculus）**: 介入の効果をモデル化（P(Y | do(X)) ≠ P(Y | X)）

### 5.3 DAG（有向非巡回グラフ）の基本

```
    SES
   ↙   ↘
出欠 ───→ 成績 ← 能力
         ↑
       動機付け
```

#### 3つのパス構造と統制の判断

| 構造 | 図 | 統制すべきか | 教育での例 |
|------|-----|-------------|-----------|
| **交絡因子（Confounder）** | D ← X → Y | **する** | SESが出欠と成績の両方に影響 |
| **媒介因子（Mediator）** | D → M → Y | **しない**（間接効果を消してしまう） | 出欠 → 理解度 → 成績 |
| **合流点（Collider）** | D → X ← Y | **しない**（統制すると偽の関連が生じる） | 出欠 → 進路選択 ← 成績 |

**バックドア基準**: 処置Dから結果Yへのバックドアパス（D ← ... → Y）をすべてブロックする変数の組を見つければ、因果効果を推定できる。

**重大な誤り**: 「統制変数は多いほど良い」は間違い。合流点（Collider）を統制すると、かえってバイアスが生じる。

### 5.4 準実験デザインの教育データへの適用

#### (1) 傾向スコアマッチング（PSM）

**原理**: 処置群（例: 補習参加者）と対照群（不参加者）を、「参加確率（傾向スコア）」が類似するもの同士でマッチングし、観測可能な交絡を除去。

**教育での応用例**: 
- カトリック学校 vs 公立学校の効果比較（家庭収入、親の教育期待、基礎学力で統制）
- 結果: ATT（処置群の平均処置効果）は約1.6ポイント（12年生の数学成績）

**手順**:
1. ロジスティック回帰で傾向スコアを推定
2. 近傍マッチング or 層別化でマッチング
3. バランスチェック（標準化平均差 < 0.1を確認）
4. マッチ後のサンプルで効果推定

**限界**: 観測されていない交絡因子（例: 教育への意欲）は統制できない

#### (2) 差分の差分法（DID）

**原理**: 介入前後の変化の「差」を、介入群と対照群で比較。

**教育データでの適用場面**:
- 新しい指導法をあるクラスで導入 → 前後の成績変化を、導入していないクラスと比較
- 前提: 介入がなければ両群は同じ傾向で変化していた（平行トレンド仮定）

```
介入群の効果 = (介入後の介入群平均 - 介入前の介入群平均)
             - (介入後の対照群平均 - 介入前の対照群平均)
```

#### (3) 回帰不連続デザイン（RDD）

**原理**: カットオフスコアによる処置割当を利用。カットオフ付近の生徒は「ほぼランダムに」処置群・対照群に分かれる。

**教育データでの適用場面**:
- 「成績がGPA 2.0以下の生徒に補習を義務化」→ GPA 1.9と2.1の生徒は能力がほぼ同じなのに、補習の有無が異なる
- イスラエルの有名な研究: クラスサイズ上限40人の政策を利用し、クラスサイズの因果効果を推定
- 奨学金の効果: テストスコアのカットオフを利用した準実験

**強み**: カットオフ付近ではRCTに近い妥当性。未観測の交絡にも頑健。
**限界**: 効果はカットオフ付近の局所的な推定値（一般化には注意）。

#### (4) 操作変数法（IV）

**原理**: 処置に影響するが、結果には直接影響しない変数（操作変数）を利用して因果効果を推定。

**教育データでの適用場面**:
- 「通学距離」を操作変数として、出席率の因果効果を推定（通学距離は出席に影響するが、成績には直接影響しない...と仮定）
- 「最寄りの大学までの距離」を操作変数として、教育年数の収入への因果効果を推定した有名な研究

**強み**: 未観測の交絡があっても因果効果を推定可能
**限界**: 「除外制約」（操作変数が結果に直接影響しない）の仮定が検証不能で、防御が困難

### 5.5 準実験デザインの選択ガイド

| 手法 | 必要条件 | 教育データでの使いやすさ | 強み |
|------|---------|----------------------|------|
| **PSM** | 交絡因子が全て観測されている | ★★★★（最も使いやすい） | 柔軟 |
| **DID** | 介入前後のデータ + 対照群 | ★★★（時系列データがあれば） | 未観測交絡に部分的に対処 |
| **RDD** | カットオフによる割当がある | ★★（条件は限定的） | RCTに近い妥当性 |
| **IV** | 適切な操作変数がある | ★（見つけるのが困難） | 最も強力 |

## 6. 実践的な「問い」のフレームワーク

### 6.1 4段階分析フレームワーク

| 段階 | 問い | 手法 | 教育データでの例 |
|------|------|------|----------------|
| **1. 記述的** | 何が起きているか？ | 集計、可視化、基本統計 | 「数学の平均点は3年間でどう推移したか」 |
| **2. 診断的** | なぜ起きたか？ | 相関分析、層別分析、クロス集計 | 「なぜ2年生で数学成績が低下する生徒が増えるのか」 |
| **3. 予測的** | 次に何が起きるか？ | 回帰モデル、機械学習 | 「この入試成績パターンの生徒は3年後どうなるか」 |
| **4. 処方的** | 何をすべきか？ | 因果推論、介入設計 | 「どの生徒にどの時点でどの介入が最も効果的か」 |

**通常は1→2→3→4の順に進む**。データ基盤が整い次第、記述的分析から始め、診断的分析で根本原因を探り、予測モデルを構築し、最終的に介入を設計する。

### 6.2 教育データから問うべき良い質問

#### 記述的レベル

- 入試成績と3年間の成績推移にはどのようなパターンがあるか？
- 出欠パターンはいつ、どのように変化するか？
- テストの設問ごとの正答率分布はどうか？
- アンケートの回答パターンにクラスタはあるか？

#### 診断的レベル

- 成績が急落する生徒に共通する事前のシグナルは何か？
- 出席率と成績の関係は、コース・クラスによって異なるか？
- 特定の設問でのつまずきは、後の設問の不正答を予測するか？
- アンケートの回答パターンと成績の関連は、入学時の学力を統制しても残るか？

#### 予測的レベル

- 入学時のデータだけで、3年後の成績をどの程度予測できるか？
- 1学期の出欠・成績パターンから、「リスクのある」生徒を早期に特定できるか？
- 模試の推移パターンから、大学入試の結果を予測できるか？

#### 処方的レベル

- 「リスクのある」生徒への早期介入は、どの時点で最も効果的か？
- つまずきの連鎖を断ち切るには、どの前提知識を補強すべきか？
- 非認知能力への介入は、どの生徒群で最も効果があるか？

### 6.3 教員の暗黙知を検証可能な仮説に変換する

#### 暗黙知の源泉

教員は7つの認知プロセス（意思決定、問題解決、推論、判断、記憶、知覚、直観）を通じて暗黙知を蓄積している。この暗黙知は「ビデオ刺激再生法」等で言語化が試みられているが、データによる検証が最も効果的。

#### 変換の3ステップ

```
Step 1: 暗黙知の言語化
  教員: 「部活をやめた生徒は成績が下がる気がする」

Step 2: 検証可能な仮説への変換
  仮説: 「部活退部後3ヶ月以内に、成績偏差値が5以上低下する割合は、
         継続者と比較して有意に高い」

Step 3: 分析設計
  方法: DID（退部前後の変化を、継続者と比較）
  変数: 退部時期、成績推移、統制変数（入学時成績、出欠、SES）
  効果量: Cohen's d + 95%CI
  多重比較: 複数の教員仮説を同時検証 → FDR制御
```

#### 教員ワークショップの設計案

1. **収集**: 教員から「感覚的に感じていること」を自由記述で収集
2. **分類**: 記述的/診断的/予測的/処方的に分類
3. **操作化**: 各「感覚」を測定可能な変数と基準に変換
4. **優先順位**: 検証可能性 × 教育的インパクト × データの可用性でランキング
5. **検証**: 分析→結果共有→解釈の議論→新たな仮説生成

## 付録: 手法の組み合わせによる分析戦略

### 推奨する分析の進め方

```
Phase 1: EDA（探索）
├── 相関行列ヒートマップ
├── クラスタリング（生徒タイプの発見）
├── 外れ値検出（非典型的パターンの発見）
├── アソシエーションルール（隠れた関連の発見）
└── 時系列パターン（成績推移の可視化）
    │
    ▼ パターン発見 → 仮説生成
    
Phase 2: 仮説検証
├── 疫学的手法（OR, RR, 層別分析）
├── 多変量回帰（交絡の統制）
├── 知識空間分析（つまずきの連鎖の特定）
├── マルチレベルモデル（クラスタリングの考慮）
└── 多重比較補正（FDR制御）
    │
    ▼ 検証済みパターン → 因果仮説
    
Phase 3: 因果推論
├── DAGの構築（因果構造の明示化）
├── PSM / DID / RDD（因果効果の推定）
└── 感度分析（結果の頑健性確認）
    │
    ▼ 因果的知見 → 介入設計
    
Phase 4: 予測と処方
├── 早期警戒モデル（リスク生徒の特定）
├── 介入効果の予測
└── 現場への知見の還元
```

### 各フェーズで使う検定力の目安

| フェーズ | 許容される偽陽性率 | 補正方法 |
|---------|-------------------|---------|
| Phase 1（探索） | 高め（10-20%） | なし or 緩いFDR |
| Phase 2（検証） | 中程度（5%） | FDR（q=0.05） |
| Phase 3（因果） | 低め（1-5%） | Bonferroni or 厳しいFDR |
| Phase 4（予測） | モデル精度で評価 | 交差検証 |
